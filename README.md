# 初中数学可视化动态演示工具集

基于AI辅助开发的初中数学教学工具，适合课堂投影演示。

## 工具列表

| 工具名称 | 功能说明 |
|---------|---------|
| 一次函数演示工具.html | 滑块调节参数k、b，实时显示直线图像、斜率意义、交点坐标 |
| 二次函数演示工具.html | 滑块调节参数a、b、c，实时显示抛物线、顶点坐标、对称轴 |
| 反比例函数演示工具.html | 滑块调节参数k，实时显示双曲线、象限位置、渐近线 |
| 勾股定理面积验证.html | 可拖动三角形，水流动画直观展示a²+b²=c²面积关系 |
| 二元一次方程组图像解法.html | 两条直线交点直观展示方程组的解 |

## 使用方法

1. 直接用浏览器打开HTML文件即可使用
2. 无需安装任何依赖
3. 适合课堂投影演示
