# 初中数学可视化动态演示工具集 - 使用说明

## 一、工具概述

本工具集包含5个初中数学可视化动态演示工具，适用于课堂教学演示。

## 二、工具详情

### 1. 一次函数演示工具

**文件**：一次函数演示工具.html

**功能**：
- 滑块调节斜率k（-5到5）和截距b（-10到10）
- 实时绘制直线图像
- 显示斜率意义（递增/递减/水平线）
- 显示y轴截距和x轴交点
- b=0时提示"正比例函数"

**教学应用**：
- 演示k的正负对直线方向的影响
- 演示b对直线上下平移的影响
- 理解一次函数与一元一次方程的关系

---

### 2. 二次函数演示工具

**文件**：二次函数演示工具.html

**功能**：
- 滑块调节参数a（-3到3）、b（-10到10）、c（-10到10）
- 实时绘制抛物线图像
- 显示开口方向（向上/向下）
- 显示顶点坐标：(-b/2a, (4ac-b²)/4a)
- 显示对称轴方程：x = -b/2a
- 显示判别式Δ及与x轴交点情况

**教学应用**：
- 演示a对开口方向和开口大小的影响
- 演示b、c对顶点位置的影响
- 理解二次函数与一元二次方程的关系

---

### 3. 反比例函数演示工具

**文件**：反比例函数演示工具.html

**功能**：
- 滑块调节参数k（-20到20）
- 实时绘制双曲线图像
- 显示k>0时图像在一、三象限
- 显示k<0时图像在二、四象限
- 虚线显示渐近线（x轴、y轴）
- 提示关于原点对称

**教学应用**：
- 演示k的正负对图像位置的影响
- 理解反比例函数的定义域和值域
- 理解渐近线概念

---

### 4. 勾股定理面积验证

**文件**：勾股定理面积验证.html

**功能**：
- 可拖动三角形顶点改变形状
- 三个正方形在三角形三边外侧
- 实时显示三边长度a、b、c
- 实时显示面积a²、b²、c²
- 水流动画：大正方形的水流向两个小正方形
- 直观展示a² + b² = c²

**教学应用**：
- 直观理解勾股定理
- 验证任意直角三角形都满足勾股定理
- 理解面积与边长的关系

---

### 5. 二元一次方程组图像解法

**文件**：二元一次方程组图像解法.html

**功能**：
- 滑块调节两条直线的参数
- 实时绘制两条直线
- 交点用红色圆点标注
- 实时显示交点坐标（方程组的解）
- 判断解的情况：唯一解、无解、无穷多解
- 预设方程组按钮

**教学应用**：
- 理解二元一次方程组的解的几何意义
- 理解两条直线位置关系与解的情况对应
- 直观展示方程组解法

---

## 三、使用方法

1. 用浏览器（Chrome、Edge、Firefox等）打开HTML文件
2. 拖动滑块调节参数
3. 观察图像实时变化
4. 适合课堂投影演示

## 四、技术说明

- 所有工具为单HTML文件，无需安装
- 使用HTML5 + CSS + JavaScript + Canvas
- 可在任何现代浏览器中运行
- 支持触摸操作（平板电脑）
